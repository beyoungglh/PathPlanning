# 相关技术

室内定位：ibeacon PDR 路径归并 滤波 BLE RFID Wifi UWB 红外 超声波 ZigBee
室外定位：GNSS GPS GLONASS BDS GALLEO 超高频无线电波

几何定位（距离交会、角度交会、航位推算）、靠近感知、场景分析

## iBeacon

iBeacon是苹果在移动设备上的功能，配备 低功耗蓝牙BLE 通信设备，每个设备向周围发送自己特有的ID。
